# URI(Uniform Resource Identifier)
> 직역하면 `유니폼 리소스 식별자`로 `리소스를 통일적으로 식별하는 ID`를 말함

### URI 구문
- URI 스펙: RFC 3986
- {URI Scheme}://{User Information}@{Host name}:{Port Number}/{Path}?{Query Parameter}#{URI Fragment}

> User Information

> Host Name

> Port Number

> Path

> Query Parameter

> URI Fragment
